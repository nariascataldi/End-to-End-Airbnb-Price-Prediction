# Introducción y Finalidad del Proyecto

## ¿Qué es este proyecto?
Este es un proyecto de **Ciencia de Datos y Machine Learning de extremo a extremo (End-to-End)** diseñado para predecir los precios de alquiler de listados de Airbnb. Utiliza datos reales para entrenar modelos de aprendizaje automático que pueden estimar el costo de una estancia basándose en diversas características de la propiedad.

## Finalidad
La finalidad principal del proyecto es proporcionar una herramienta que ayude tanto a **anfitriones** como a **huéspedes**:
- **Para Anfitriones:** Optimizar sus ganancias estableciendo precios competitivos basados en datos reales del mercado, en lugar de intuiciones.
- **Para Huéspedes:** Encontrar ofertas justas y entender qué factores influyen en el precio de un alojamiento.

## Objetivos Técnicos
1. **Recolección de Datos:** Ingesta de datos desde fuentes externas (Kaggle).
2. **Procesamiento de Datos:** Limpieza y transformación de datos brutos en un formato apto para modelos de ML.
3. **Entrenamiento de Modelos:** Comparación de diferentes algoritmos (Linear Regression, Lasso, Ridge, Random Forest, XGBoost, CatBoost, AdaBoost) para encontrar el mejor predictor.
4. **Implementación de Pipeline:** Creación de un flujo de trabajo automatizado para la predicción.
5. **Interfaz de Usuario:** Una aplicación web simple desarrollada con **Flask** para que cualquier usuario pueda ingresar datos y obtener una predicción instantánea.

## ¿Por qué es importante?
En el mercado dinámico de Airbnb, factores como la ubicación, el tipo de habitación, la cantidad de reseñas y las comodidades (amenities) influyen drásticamente en el precio. Este proyecto desglosa esa complejidad mediante el poder del análisis de datos.
